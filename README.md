# TuPrimeraPagina-Peries
Tercera entrega - Coderhouse.
Gestión de Clientes y Productos
Permite gestionar clientes, productos y pedidos.

Funcionalidades
Clientes
Funcionalidades:
Registrar cliente ---> /cliente/
Ver lista de clientes ----> /clientes/
Buscar cliente por nombre (desde la lista) ----> /clientes/?nombre=...
Buscar cliente (formulario dedicado) ----> /buscar/

Productos
Funcionalidades:
LRegistrar producto ---> /crearproducto/
Ver lista de productos ---> /productos/
Buscar producto por nombre ----> /productos/?nombre=...
Pedidos
Funcionalidades: 
Crear pedido ----> /crearpedido/
Ver lista de pedidos ----> /pedidos/
Buscar pedidos por nombre de cliente ----> /pedidos/?cliente=...
